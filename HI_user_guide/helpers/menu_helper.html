Menu Helper

The Menu Helper file contains functions that assist in deploying pre-loaded rollover menus. It establishes and manipulates the global variable $menus, which you can use to hard-wire the preloading and display where you'd like it.

Loading this Helper

This helper is loaded using the following code:

$this->load->helper('menu');

The following functions are available:

init_stylish_buttons()

Returns <HEAD> code that deploys the Adobe / Macromedia preloading rollovers code.

Example:

$page->append_head( init_stylish_buttons() );

onload_preload_images()

Returns code to be added to your onload tag attribute. Receives an array of urls to images that are to be preloaded.

$page->add_onload( onload_preload_images( array( "someimage.jpg", "someotherimage.png" ) );

make_menu_horiz( name, path_to_images, items_array, filler_image, trailing_w )

Generates the content that displays a simple menu in a horizontal fashion. The parameter trailing_w is width of the filler image at the end of the menu. This function returns the name of the menu and updates the global $menus array.

$items=array();
$items[] = array( 0=>here().'/home', 1=>'View your home wall.', 2=>'wall' );
$items[] = array( 0=>here().'/photos', 1=>'View and manage your photos.', 2=>'photos' );
$items[] = array( 0=>here().'/friends/of/'.$this->username, 1=>'See who are your friends.', 2=>'friends' );
$items[] = array( 0=>here().'/blog/by/'.$this->username, 1=>'Read or write your blog!', 2=>'blog' );
$items[] = array( 0=>here().'/account/edit/'.$this->username, 1=>'Edit your profile settings.', 2=>'edit' );
$fillwidth=146;

make_menu_horiz( "profile", base_url().'MENU', $items, base_url().'MENU/filler.png', $fillwidth );

$menus = get_menus();
$page->append_head( $menus['head'] );
$page->add_onload( $menus['onload'] );
$page->append_body( $menus['profile'] );

make_menu_vert( name, path_to_images, items_array )

Generates the content that displays a simple menu in a horizontal fashion. The parameter trailing_w is width of the filler image at the end of the menu. This function returns the name of the menu and updates the global $menus array.

$items=array();
$items[] = array( 0=>here().'/home', 1=>'View your home wall.', 2=>'wall' );
$items[] = array( 0=>here().'/photos', 1=>'View and manage your photos.', 2=>'photos' );
$items[] = array( 0=>here().'/friends/of/'.$this->username, 1=>'See who are your friends.', 2=>'friends' );
$items[] = array( 0=>here().'/blog/by/'.$this->username, 1=>'Read or write your blog!', 2=>'blog' );
$items[] = array( 0=>here().'/account/edit/'.$this->username, 1=>'Edit your profile settings.', 2=>'edit' );
$fillwidth=0;

make_menu_vert( "profile", base_url().'MENU', $items, base_url().'MENU/filler.png' );

$menus = get_menus();
$page->append_head( $menus['head'] );
$page->add_onload( $menus['onload'] );
$page->append_body( $menus['profile'] );
