JS Helper

The JavaScript Helper file contains functions that assist in working with basic javascript.

Loading this Helper

This helper is loaded using the following code:

$this->load->helper('js');

The following functions are available:

js( code )

Simply wraps some Javascript in a <SCRIPT> tag.

Example:

$page->append_head( js ( "var a=5;" ) );

add_event( jsid, event_attribute, code )

Uses Javascript to add events to an object with an existing Javascript event handler.

Usage example:

$page->append_head( add_event( "domid", "onchange", "javascript:this()" ) );

add_event_id( jsid, event_attribute, code )

Append an event to an existing Javascript event handler on given object (usually a div with an id).

Usage example:

$page->append_head( add_event_id( "domid", "onchange", "javascript:this()" ) );

alert( message )

Triggers a Javascript alert.

Usage example:

$page->append_head( alert( "Alert! Alert! All hands to battlestations!" ) );

jsredirect( url, delay_in_millis )

Javascript redirect with delay.

Usage example:

$page->append_body( jsredirect( "http://foofle.com", 20 ) );

add_onload( code )

Also a part of the Page class as $this->page->add_onload( "somefunction()" ), uses JavaScript to append a function to the onload attribute.

Usage example:

add_onload( "javascriptfunction()" );

js_character_counter( size="255", dom_id="text", div_id="counter" )

Establishes a character counter in a div with its own id (counter) attached to an input or textarea tag with an id (text)

Usage example:

$page->append_body( js_character_counter( 1024, "message", "msg_counter") );

setup_tinymce( path=here()."/js" )

Initializes tinyMCE. Should be called in the <HEAD> section.

Usage example:

$page->append_head( setup_tinymce( ) );

tinymce( $name = "elm1", $content="" )

Also a part of the Page class as $this->page->add_onload( "somefunction()" ), uses JavaScript to append a function to the onload attribute.

Usage example:

$page->append_body( tinymce("my_text_id","some default content" ) );

use_zorng( path=here()."/js" )

Initializes Walter Zorn's VectorGraphics library for JavaScript.

Usage example:

$page->append_head( use_zorng( ) );

wz_start( ) ... wz_end( ), or wz_draw( jsid, code )

Begins a ZornG drawing codeblock. Terminate this block with wz_end();

Activates these wrappers around the ZornG VG functions:

fillRect( $left, $top, $width, $height )
drawRect( $left, $top, $width, $height )
setColor ( $color )
drawText( $content, $x, $y )
textRect( $content, $x, $y, $w, $alignment="left" /* or "right" */ )
stroke( $thickness )
line( $x, $y, $xx, $yy )
jghtm( $html_insert )
polyline( $array1, $array2 )
polygon( $array1, $array2 )
fillPolygon( $array1, $array2 )
oval( $x, $y, $xx, $yy )
fillOval( $x, $y, $xx, $yy )
ellipse( $x, $y, $xx, $yy )
fillEllipse( $x, $y, $xx, $yy )
arc( $x, $y, $xx, $yy, $sa, $ea )
fillArc( $x, $y, $xx, $yy, $sa, $ea )
clear( )
setFont( $name = "verdana,geneva,sans-serif", $size = "10px", $drawstyle = "PLAIN" /* or BOLD, ITALIC, BOLD_ITALIC */ )
newCanvas( $name = "myCanvas", $zg_height = "100%", $zg_width = "100%" )

Usage examples:

// Draws while the page is loaded.
$page->append_head(
wz_start() .
setColor("#00ff00") .
fillEllipse(100, 200, 100, 180) .
setColor("maroon") .
drawLine(10, 113, 220, 55) .
setColor("#0000ff") .
fillRect(110, 120, 30, 60) .
setColor("#0000ff") .
drawEllipse(10, 50, 30, 100) .
drawRect(400, 10, 100, 50).
wz_end() );

// You can also wrap it inside a JS function to be called on demand.
$page->append_head(
wz_draw(
setColor("#00ff00") .
fillEllipse(100, 200, 100, 180) .
setColor("maroon") .
drawLine(10, 113, 220, 55) .
setColor("#0000ff") .
fillRect(110, 120, 30, 60) .
setColor("#0000ff") .
drawEllipse(10, 50, 30, 100) .
drawRect(400, 10, 100, 50)
) // end wz_draw block
); // end append_head

$page->append_body( "<a href='#' onclick='javascript:ourfunc()'>" );
