Social Helper

The Social Helper file contains functions that assist in working with other websites such as Google and Facebook, and provides basic social web features like tag clouds.

Loading this Helper

This helper is loaded using the following code:

$this->load->helper('social');

The following functions are available:

track_with_ga( tracker_id )

The required code to let Google track your webness.

Example:

$page->append_head( track_with_ga( "Your Google Analytics Code" ) );

meta_tagger( title, image_full_url, description, keywords='' )

Makes Facebook-compatible meta tags.

Usage example:

$page->append_head( meta_tagger( "This Page Title", here().'/images/this_page_thumb.jpg', "A great page, come visit.", "page visit thumb title" ) );

tag_cloud( tags, search_url_prefix )

Generates a tag cloud that links to your desired search algorithm.

Usage example:

// send it to a search object in your site
$page->append_body( tag_cloud( array( "alpha", "beta", "gamma", "delta" ), here().'/search/' ) );

// or use URL variables
$page->append_body( tag_cloud( array( "alpha", "beta", "gamma", "delta" ), "http://www.google.com?keyword=" ) );
